# Second Brain — Autonomous Knowledge Management Skill

Second Brain — это полностью автономный, оффлайн-менеджер знаний, работающий на базе методологии **PARA** (Projects, Areas, Resources, Archives). Навык организует ваши заметки, выполняет семантический поиск через векторные эмбеддинги и управляет локальной персональной базой данных на устройстве.

## Основные возможности

- **PARA-методология:** Автоматическая категоризация знаний по четырём категориям
- **Векторный поиск:** Семантический поиск по смыслу, а не по ключевым словам
- **100% Оффлайн:** Все данные хранятся локально, без отправки на сервер
- **Интеграция с системой:** Создание контактов, напоминаний, событий календаря
- **Аудио-ввод:** Поддержка голосовых заметок с автоматической расшифровкой
- **Самоэволюция:** Агент может адаптировать свой код при необходимости

## Системные инструкции

Агент использует следующие правила:

### Категоризация (PARA)
- **Projects:** Временные инициативы с четким дедлайном
- **Areas:** Сферы ответственности (здоровье, финансы, развитие)
- **Resources:** Справочные материалы и знания
- **Archives:** Завершённые проекты и неактуальная информация

### Доступные инструменты
1. `save_to_para` — сохранение информации с векторными эмбеддингами
2. `retrieve_memory` — семантический поиск по БД
3. `create_calendar_event` — создание событий в календаре
4. `create_contact` — создание контактов
5. `set_alarm` — установка будильников
6. `send_sms` — отправка SMS
7. `device_control` — управление системными функциями
8. `evolve_code` — адаптация JavaScript-скрипта при ошибках

## Архитектурные требования

- **Persistent Memory:** SQLite WASM с IndexedDB для асинхронного сохранения
- **Гибридный RAG:** Оптимизирован для on-device среды
- **Изоляция инструментов:** Все вызовы перехватываются через глобальный асинхронный обработчик
- **Zero External Dependencies:** Только встроенные библиотеки, никаких CDN
- **SQL Security:** Параметризация всех запросов, защита от инъекций
- **Explicit DB Handlers:** Каждый инструмент должен иметь обработчик в коде
